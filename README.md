# 📊 VendorPulse Analytics

## **🔹 Overview**
VendorPulse Analytics is an end-to-end data analysis project focused on evaluating vendor performance, inventory efficiency, and profitability using **Python, SQL, and Power BI**.

---

## **🎯 Objective**
- Identify underperforming brands  
- Analyze top vendors and their contribution  
- Optimize pricing and bulk purchasing strategies  
- Improve inventory turnover  

---

## **🛠️ Tech Stack**
- **Python** – Data cleaning & EDA  
- **SQL** – Data querying & transformation  
- **Power BI** – Dashboard & visualization  

---

## **📂 Project Structure**
VendorPulse-Analytics/
│
├── data/ # Raw & processed datasets
├── sql/ # SQL queries
├── python/ # Data analysis scripts
├── dashboard/ # Power BI files (.pbix)


---

## **📈 Key Insights**
- 198 brands show **low sales but high profit margins**  
- Top 10 vendors contribute **65%+ of purchases**  
- Bulk purchasing reduces unit cost by **~72%**  
- Unsold inventory worth **$2.71M** affects efficiency  

---

## **📊 Outcome**
Generated actionable insights to improve vendor performance, reduce dependency risks, and optimize inventory management.
