📊 VendorPulse Analytics
🔹 Overview

VendorPulse Analytics is an end-to-end project that analyzes vendor performance, inventory efficiency, and profitability using Python, SQL, and Power BI.

🎯 Objective

Identify underperforming brands

Analyze top vendors and their contribution

Optimize pricing & bulk purchasing

Improve inventory turnover

🛠️ Tech Stack

Python (EDA, data cleaning)

SQL (data querying)

Power BI (dashboard & visualization)

📈 Key Insights

198 brands have low sales but high margins

Top 10 vendors contribute 65%+ purchases (high dependency)

Bulk buying reduces cost by ~72%

Unsold inventory worth $2.71M impacts efficiency
